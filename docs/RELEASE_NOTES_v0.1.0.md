# Light Pilot v0.1.0

Initial public preview.

## Highlights

- Tray-first Windows screen comfort app.
- Local-only settings and optional local content brightness analysis.
- Gentle adaptive brightness and warmth.
- Game/video fullscreen protection.
- Simple main UI with practical feedback buttons.
- Local install, uninstall, smoke, and packaging scripts.

## Known Limits

- DDC/CI support depends on monitor, cable, dock, GPU driver, and firmware.
- Software overlay changes perceived warmth but does not reduce panel backlight power.
- App override editing is intentionally simple text in Advanced settings.
